# Crypto Quant v4.0 — Universal Trading Strategy

과적합 없이 **12개 암호화폐에서 범용적으로 작동**하는 퀀트 트레이딩 전략 + 실시간 대시보드

## 전략 특징

| 항목 | 내용 |
|:---|:---|
| 진입 조건 | EMA(21) 추세 + MACD(12/26/9) 모멘텀 + 거래량 확인 |
| 청산 방식 | **순수 ATR 트레일링 스탑** (최고가 - 2×ATR) |
| 조정 파라미터 | **1개** (ATR 배수, 기본 2.0) |
| 데이터마이닝 수치 | **0개** — 모든 파라미터가 업계 표준 디폴트 |

### 왜 이 전략인가?

기존 v2/v3 전략은 StochRSI, 볼린저밴드, ADX 등 다수의 지표 기반 청산 조건을 사용했으나, 과적합 검증 결과 **특정 임계값(StochRSI 95 등)이 과거 데이터에 과적합**되어 있음을 확인:

- v3: 새 코인 적용 성공률 25%, 파라미터 민감도 높음
- **v4: 새 코인 적용 성공률 83%, 파라미터 안정적 (1.5x~3.0x 모두 수익)**

## 파일 구성

```
universal_strategy.py     # v4 범용 전략 백테스트 (v2/v3/v4 비교 포함)
overfit_validation.py     # 과적합 검증 (Walk-Forward, OOS, 파라미터 민감도)
live_dashboard_v3.py      # 실시간 웹 대시보드 (6개 코인 동시 모니터링)
requirements.txt          # 의존성
```

## 설치 및 실행

```bash
pip install -r requirements.txt
```

```bash
# 백테스트 실행 (12개 코인, v2/v3/v4 비교)

# 과적합 검증
python overfit_validation.py

# 실시간 대시보드
python live_dashboard_v3.py
```

## 대시보드 기능

- 6개 코인 실시간 모니터링 (BTC, ETH, SOL, XRP, ADA, DOGE)
- ADX 기반 시장 체제 표시 (BULL / NORMAL)
- 트레일링 스탑 상태 실시간 추적
- 지표 게이지 (ADX, StochRSI, BB, Volume, MACD)
- 거래 내역 로그
- SSE 기반 실시간 업데이트

## 주의사항

- **과거 성과는 미래 수익을 보장하지 않습니다**
- 실제 거래에는 슬리피지, 수수료, 심리적 요인이 추가됩니다
- 백테스트 수익의 30~50% 수준으로 기대치를 잡으세요
- 한 코인에 올인하지 말고 분산 투자를 권장합니다
- 이 프로젝트는 교육/연구 목적이며, 투자 조언이 아닙니다
