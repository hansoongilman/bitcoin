# 🚀 Auto Bitcoin - 자동 암호화폐 거래 봇

안녕하세요! 이 프로젝트는 **여러 암호화폐를 자동으로 매매하는 지능형 거래봇**입니다.
단순한 자동화를 넘어, 기술적 분석을 기반으로 한 **스마트한 매매 전략**을 사용합니다.

> 💡 **핵심 아이디어**: ADX 기반 이중 체제로 상승장과 보통장을 구분하여 각 상황에 맞는 최적의 전략을 실행합니다.

---

## 📊 주요 기능

### 🤖 자동 매매 시스템
- **6개 코인 모니터링**: BTC, ETH, SOL, XRP, ADA, DOGE
- **V3 개선 전략**: ADX 기반 이중 체제로 더욱 정교한 의사결정
- **실시간 트레일링 스탑**: 최고점을 자동 추적하며 손실을 최소화
- **빠른 재진입**: 좋은 기회를 놓치지 않고 즉시 재진입

### 📈 백테스팅 도구
- 과거 데이터로 전략의 수익성 검증
- 다양한 시간대 및 시장 조건 분석
- 매개변수 튜닝을 통한 최적화

### 🎨 라이브 대시보드
- 웹 기반 실시간 모니터링 인터페이스
- 각 코인별 거래 상태, 수익률, 기술적 지표 시각화
- 프리미엄 다크 테마 UI로 편안한 사용 경험

## 🧠 V3 전략의 비법

### 핵심 개선사항

```
1️⃣ ADX 이중 체제
   - ADX > 25 && DI+ > DI- → 강세 모드 활성화
   - 강세 모드에서 더 적극적인 거래 실행

2️⃣ 스마트 청산 조건
   - 강세 모드: StochRSI 임계값 85 → 95로 상향
   - MACD 역전 무시 (더 긴 트렌드 추적)
   - ATR 기반 트레일링 스탑으로 손실 최소화

3️⃣ EMA 트레일링 스탑
   - 10주기 EMA 기반 동적 손절
   - 2xATR 거리로 유연한 마진 유지

4️⃣ 빠른 재진입
   - 쿨다운 없이 즉시 새로운 신호 대응
```

## 🎯 사용 시나리오

1. `improved_strategy_v3.py` 코드 분석

### 🧪 전략을 검증하고 싶을 때
1. `enhanced_strategy_backtest.py` 실행
2. 과거 데이터로 수익률, MDD(최대낙폭) 확인
3. `param_tuning_backtest.py`로 최적 매개변수 찾기

### 📊 실시간 모니터링
1. `live_dashboard_v3.py` 실행
2. 웹 브라우저에서 대시보드 오픈
3. 각 코인의 진입/청산 신호 실시간 확인

## 📈 예상 성능

- **수익률**: 과거 데이터 기반 월 5~15% (시장 상황에 따라 변동)
- **최대낙폭(MDD)**: 8~12%
- **승률**: 55~65%

> ⚠️ **과거 성과는 미래 수익을 보장하지 않습니다.**
> 항상 소액으로 시작하여 검증 후 확대하세요!

## 📝 자주 묻는 질문

**Q: 실제 자동매매에 바로 쓸 수 있나요?**
> A: 아직입니다. 더 많은 테스트와 실전 검증이 필요합니다. 먼저 백테스트로 신뢰성을 확인하세요!

**Q: 어떤 시장 상황에서 잘 작동하나요?**
